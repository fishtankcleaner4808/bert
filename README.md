# Bert, Phase 1

This is the Phase 1 refactor of the Bert SDL2 platformer engine.

## Included

- Multi-file architecture
- Separate game systems
- Player module
- Enemy module
- Tilemap module
- Camera module
- Render module
- Fixed timestep physics
- Interpolated rendering
- Camera smoothing
- Menu system

## Build

```bash
make
```

## Run

```bash
./bert
```

## Required Assets

Place these files into:

```text
assets/
```

Required:
- BYTE.TTF
- map.txt
